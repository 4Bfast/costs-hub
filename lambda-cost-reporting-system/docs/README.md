## Getting Started

### For New Client Onboarding
1. Start with the [Client Onboarding Guide](client-onboarding-guide.md)
2. Use [AWS IAM Policy Templates](aws-iam-policy-templates.md) to set up client permissions
3. Refer to [Troubleshooting Guide](troubleshooting-guide.md) for common issues

### For System Administration
1. Review the main README.md in the project root for setup instructions
2. Follow [Deployment](deployment.md) guide for infrastructure setup
3. Configure [Monitoring](monitoring-setup.md) and [Dashboards](monitoring-dashboards.md)

### For Development
1. See [Local Development](local-development.md) setup guide
2. Review implementation summaries for understanding system components
